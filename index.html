EdgeRobotArm by kyllikki

Edge robot arm software

The "Edge" Robot ARM is a USB controlled robot arm. The arm is also sold under a number of of other names from numerous retailers including from Maplin

Dependencies

The library depends on libusb version 1.0 The UI depends on vala version 0.7.6

Install

Building the software --------------------- It can be built with a simple make invocation: $ make This builds both the library "libedgerbtarm.so" and the user interface "edgerbtarm" If you want to execute "edgerbtarm" from the build directory the LD_LIBRARY_PATH needs setting or the error "./edgerbtarm: error while loading shared libraries: libedgerbtarm.so: cannot open shared object file: No such file or directory" will occur. $ LD_LIBRARY_PATH=$PWD ./edgerbtarm If the user executing the program does not have access to the USB device or it is not plugged in an error will be displayed "Unable to connect to USB device" but the program will not exit.
